# Bank Campaign Business Analysis — Power BI Report

## Project Overview
This project analyzes a bank's telemarketing campaign data to identify conversion opportunities and operational inefficiencies. Built as a Business Analyst portfolio project using Power BI.

## Business Questions
- Which customer segments have the highest conversion rates?
- Which months yield the best campaign performance?
- Where is the bank wasting outreach effort on low-probability leads?

## Tools & Skills
- **Power BI Desktop** — report building, data modeling
- **Power Query** — data cleaning, null handling, custom columns
- **DAX** — calculated measures (Conversion Rate, Total Subscribed, Avg Call Duration)

## Dataset
- Source: [UCI Machine Learning Repository — Bank Marketing Dataset](https://archive.ics.uci.edu/dataset/222/bank+marketing)
- 41,188 records, 21 features
- Real-world telemarketing campaign data from a Portuguese bank

## Key Findings
- Students and retired customers convert at 30%+, yet represent a small share of total contacts
- March and December campaigns yield significantly higher conversion rates
- The bank should reallocate outreach toward high-converting segments
